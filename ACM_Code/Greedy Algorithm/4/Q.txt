奉太郎邀请n位古典文学部的社员来玩一个游戏。首先，他让每个社员在左、右手上面分别写下一个整数，奉太郎自己也在左、右手上各写一个整数。然后，让这n位社员排成一排，奉太郎站在队伍的最前面。排好队后，所有的社员都会获得奉太郎给的若干游戏币，每位社员获得的数量分别是：排在该社员前面的所有人（注意是所有人，所以当然包括奉太郎）的左手上的数的乘积除以他自己右手上的数，然后向下取整得到的结果。
 
奉太郎不希望某一个社员获得特别多的游戏币，
所以他想请你帮他重新安排一下队伍的顺序，
使得获得游戏币最多的社员，所获游戏币尽可能的少。
注意，奉太郎的位置始终在队伍的最前面。

此题无需高精度

输入格式

第一行包含一个整数 n (1≤n≤100)n\ (1 \le n \le 100)n (1≤n≤100)，表示社员的人数。
第二行包含两个整数 a,ba,ba,b，之间用一个空格隔开，分别表示奉太郎左手和右手上的整数。
接下来n行，每行包含两个整数 a,ba,ba,b，之间用一个空格隔开，分别表示每个社员左手和右手上的整数。

输出格式

输出保证在 int32_t 范围内
输出只有一行，包含一个整数，表示重新排列后的队伍中获奖赏最多的社员所获得的金币数。

样例数据

输入

3 
1 1 
2 3 
7 4 
4 6 

输出

2


备注

输入输出样例说明
按1、2、3号这样排列队伍，获得游戏币最多的社员所获得金币数为2； 
按1、3、2这样排列队伍，获得游戏币最多的社员所获得金币数为2；
按2、1、3这样排列队伍，获得游戏币最多的社员所获得金币数为2；
按2、3、1这样排列队伍，获得游戏币最多的社员所获得金币数为9；
按3、1、2这样排列队伍，获得游戏币最多的社员所获得金币数为2； 
按3、2、1这样排列队伍，获得游戏币最多的社员所获得金币数为9。
 
因此，游戏币最多的社员最少获得2个金币，答案输出2
