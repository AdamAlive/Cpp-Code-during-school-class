    你有 N(1≤N≤1000)N (1\leq N \leq 1000)N(1≤N≤1000) 盏灯，它们排成一排，从左到右，依序标号为 1,2,3,?,N1,2,3,\cdots,N1,2,3,?,N。
    一开始，一部分的灯是亮着的，一部分的灯是灭着的。

输入格式

单个文件中包含了多组数据，请循环读取至末尾。
每组数据是一个字符数不超过 100010001000 的字符串，独占一行，由 'N' 和 'Y' 组成，从开头到结尾依序表示第 1,2,3,?,N1,2,3,\cdots,N1,2,3,?,N 号灯的开关状态，'N' 表示熄灭，'Y' 表示点亮。

输出格式

对于每组数据，如果可以通过操作使得所有灯熄灭，请输出一个整数表示最小的操作次数，独占一行。否则请输出 ?1-1?1，独占一行。
