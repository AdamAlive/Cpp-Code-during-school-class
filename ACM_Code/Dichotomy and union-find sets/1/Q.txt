题目描述

司马dalao正在调查司马王国的交通状况，得到现有司马王国道路统计表，表中列出了每条道路直接连通的城市。司马dalao的目标是使全国任何两个城市间都可以实现交通（但不一定有直接的道路相连，只要互相间接通过道路可达即可）。问最少还需要建设多少条道路？

输入格式

测试输入包含若干测试用例。每个测试用例的第1行给出两个正整数，分别是城市数目N ( < 1000 )和道路数目M；随后的M行对应M条道路，每行给出一对正整数，分别是该条道路直接连通的两个城市的编号。为简单起见，城市从1到N编号。
注意:两个城市之间可以有多条道路相通,也就是说
3 3
1 2
1 2
2 1
这种输入也是合法的
当N为0时，输入结束，该用例不被处理。

输出格式

对每个测试用例，在1行里输出最少还需要建设的道路数目。

样例数据

输入

4 2
1 3
4 3
3 3
1 2
1 3
2 3
0 

输出

1
0
