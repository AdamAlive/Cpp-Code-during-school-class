题目描述

众所周知，林大佬是司马大佬研制的AI。司马大佬喜欢在每周五晚上和他的AI玩一个有趣的游戏。司马大佬将学院里所有的x只白猫和y只黑猫放进一个袋子里，两个人轮流从袋子中抓出一只猫（林大佬先抓）。由于司马大佬的抓猫方式过于残暴，他的每一次抓取都会让袋子中的猫受到惊吓，袋子中的一只猫（白猫或黑猫）会逃出袋子。由于林大佬深知猫的特性，他的抓取不会惊吓到袋子里的猫。(每只猫逃出袋子和被抓取的概率相同)
获胜条件

    第一个抓出白猫的人获胜。
    假如袋子中没有猫了，且两个人都没有抓到白猫，则司马大佬获胜。

尽管林大佬是一个AI，懒惰的他希望你能帮助他计算一下他获胜的概率是多少？

输入格式

本题输入包括多组数据。
每组数据输入，包括两个整数xxx和yyy(0?≤?x,?y?≤?1000)。

输出格式

对于每组数据，输出一行小数（即林大佬获胜的概率），结果保留五位小数。

样例数据

输入

1 3
5 5

输出

0.50000
0.65873
