题目描述

小a有一个数X，她想把这个数变成Y,他只有两种操作，因为变过去就不能变回来，所以小a一直在小心翼翼地操作着，作为一名绅士，你能帮帮她吗？出于礼貌，如果能从X变成Y，还需要你算一下步数呢~

哎呀，忘了说了，操作有两种

1.将这个数乘上2，即 x->2*x

2.在这个数后面加上“1”，即x->10*x+1

还需要注意的是以上两种操作都可以进行无线次哟，呵呵呵

还需要更加注意的是小a一开始以为拿到这个X也算是1步，所以请把你的答案加上1哟，不然会有不可描述的错误

输入格式

两个数，分别表示x,y.（x,y<=1,000,000,000)

输出格式

如果不能从x变成y，请输出“I am so sorry.”

否则，请输出“Yes,it takes XX steps.” 这里XX表示所需的最小步数+1。

样例数据

输入

2 162

输出

Yes,it takes 5 steps.
