题目描述

随随便便算出答案后，你发现原来是三个坐标，“这个大J又要干啥？”你百思不得其解，忽然，一页纸片又落了下来，上面写道：

"很聪明嘛，最后，我就大发慈悲地告诉你，你想要知道的地点是(x,y)，而且这个图标和你算出的三个点正好组成一个正方形哟~。想知道这背后所有的秘密吗？去算吧，去算吧，去算吧，哇咔咔咔咔咔咔卡！"

奇怪的笑声仿佛依旧弥漫在空气中"可恶，我最差的就是几何了！"，但现在！无疑就是你的表演时间！（当当当当！！！）

输入格式

输出有三行，表示你上题算出来的3个坐标

输出格式

输出有一行，“The final location is (x,y)!”,这里(x,y)表示你的答案

样例数据

输入

0 0
1 0
0 1

输出

The final location is (1,1)!


备注

为了简单，这里的正方形特指四条边与坐标轴平行的正方形

此外，大J很耿直，这个地点一定存在哟
