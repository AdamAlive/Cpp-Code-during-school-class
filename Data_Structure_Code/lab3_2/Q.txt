2.  n number of persons form a circle. The first person starts to count 1. The next person count 2 and so on. The person who counts t will leave the circle. This game will end if no person remains in the circle. 
 
 Write a simple program to output the sequence of persons leaving the circle. The input data contains two lines. The first line stores the value of t and the second line stores the value of n.  
 
Example: Input:
 2 5 
 Output: 2 4 1 5 3
